# NekoXray / NekoBox For PC

Qt based cross-platform GUI proxy configuration manager (backend: Xray / sing-box)

Support Windows / Linux out of the box now.

基于 Qt 的跨平台代理配置管理器 (后端 Xray / sing-box)

目前支持 Windows / Linux 开箱即用
